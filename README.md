 # SEAL – Software Engineering Hackathon Management System ( Hệ thống quản lý cuộc thi SEAL Hackathon ngành Kỹ thuật Phần mềm).

## Thông tin dự án

* Tên đề tài: SEAL – Software Engineering Hackathon Management System
* Nhóm: Java

## Mô tả
SEAL (Software Engineering Agile League) là cuộc thi Hackathon học thuật thường niên dành cho sinh viên ngành Kỹ thuật Phần mềm.
Hệ thống được xây dựng nhằm hỗ trợ quản lý toàn bộ quá trình tổ chức cuộc thi từ đăng ký đội thi, quản lý vòng thi, nộp bài dự thi, chấm điểm, xếp hạng cho đến công bố kết quả.

* Quản lý sự kiện Hackathon
* Quản lý vòng thi
* Quản lý hạng mục thi đấu
* Quản lý đội thi
* Quản lý Mentor và Judge
* Nộp bài dự thi
* Chấm điểm và xếp hạng
* Trao giải và công bố kết quả
